# Quran Companion

A modern Quran Companion web application built with React.js and Tailwind CSS. It allows users to read the Holy Quran, view prayer times, access daily duas, use a digital tasbih, and find the Qibla direction.

## Features

- 📖 Read all Quran Surahs
- 🔍 Search Surahs
- 📑 Surah Details
- 🕌 Prayer Times
- 🤲 Daily Duas
- 📿 Digital Tasbih
- 🧭 Qibla Direction
- 🌙 Dark / Light Theme
- 📱 Fully Responsive Design

## Technologies Used

- React.js
- Vite
- JavaScript
- Tailwind CSS
- React Router DOM
- Axios
- Al Quran Cloud API
- Local Storage

## Installation

```bash
git clone https://github.com/Sadaf-2/Quran-companion.git
cd Quran-companion
npm install
npm run dev
```

## Build

```bash
npm run build
```
